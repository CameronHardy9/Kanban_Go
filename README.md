# Kanban Go

### Simple kanban app for project management

## Features
- MERN Stack
- User Auth
- New User Creation
- Server Using MongoDB
- Caching
- Multiple User Projects
- Auto Save
- Save Status Indicator
- Drag & Drop
- Add/Edit/Delete Options for Tasks, Columns, and Projects
